# Vibe Prompts

## Code review
```
can you code review this and check for breaking bugs or crucial breaking points?
find root causes and suggest 1-2 solutions depending on the complexity.
make sure to not touch irrelevant parts of the code or break things that you might lack context for.
Do not write any code, just respond back.
```

## Suggest solutions
```
give me two suggested solutions how we can improve this and simplify and follow clean code conventions. startup prototype 90/10 impact to effort
```

## Clarify
```
Ask me any clarifying questions for facts or details that you must know before proceeding confidently. Do not write any code, just respond back.
```
